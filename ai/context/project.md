# Project Context

## Name
Elvin UI (ou ton nom final)

## What this is
A Nuxt 4 UI kit + template system with copy-paste components and animated UI blocks.

## Core idea
- Not a package
- Not an installable library
- A copy-paste UI system
- Focus on speed + developer experience

## Target users
- Nuxt developers
- Freelancers
- Indie hackers
- Startup devs

## Main value
- ready-to-use components
- beautiful UI out of the box
- GSAP-based micro-interactions
- minimal setup (Tailwind required)

## Rules of the product
- Everything must be copy-paste friendly
- No hidden dependencies
- Code must be readable and modifiable
- Animation must be subtle, not heavy

## Philosophy
"Don’t install. Just copy. Build faster."
