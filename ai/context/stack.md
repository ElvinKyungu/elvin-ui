# Tech Stack

## Frontend

- Nuxt 4
- TypeScript
- TailwindCSS
- GSAP (animations)

## Code style

- script setup only
- composition API
- clean TypeScript typing
- no over-engineering

## UI principles

- minimal design
- strong spacing system
- consistent variants (primary, secondary, ghost)
- focus on usability over complexity

## Animation rules

- GSAP preferred for interactions
- micro-interactions only
- no heavy page transitions
- keep performance smooth

## Constraints

- no backend in core library
- no auth system in MVP
- no external UI frameworks

## Dependency rule

- everything must be self-contained or explicitly documented
